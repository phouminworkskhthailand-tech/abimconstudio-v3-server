🔎 Security Checklist

✓ Challenge-Response — Every login requires a one-time nonce from /api/challenge

✓ Rate Limiting — 5 attempts/60s → 5-min lockout; 15+ total → 1-hour lockout

✓ HWID Binding — Device fingerprint bound per license, max devices enforced

✓ JWT Expiry — Session tokens expire in 8 hours; client uses token-expiry auth

✓ Input Validation — Email and license key format validated server-side

✓ No Offline Bypass — Hardcoded offline licenses removed from extension

✓ R2 Signed URLs — Download links expire in 5 minutes, HWID verified

✓ DevTools Detection — Login dialog locks when browser inspector is open
