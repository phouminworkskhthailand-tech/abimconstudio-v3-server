Everything you need to estimate from a SketchUp model.

📐

BOQ Input

Tag any component, layer or material with a 52-column BOQ schema. Per-role units, factors, wasted % and currency are all editable.

📊

Figma-style Report

Drag-and-drop materials, cost codes and labour onto rows. Page-break preview. Export to Excel, PDF, or save to cloud.

🤖

AI Assistant

Ask the BOQ AI about your model: cost breakdowns, line-item descriptions, BIM auto-summary. Credit-metered, never blocks UX.

🧊

3D BIM Library

Pick from a Cloudflare R2 catalog of vetted SKP families. One-click attach — the model imports cleanly with quantities preserved.

☁️

Cloud Sync

Save the whole report (tabs, margins, items) to your account. Reload from any machine; multiple device limit per license.

🌍

Multilingual

Link items across languages. Switch a project from English to Thai or Vietnamese without re-modeling. Region/standard fields built-in.
